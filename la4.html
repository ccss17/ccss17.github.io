벡터 공간과 부분 공간

벡터 공간(vector space) : 벡터의 집합이다.

  • 벡터 공간에는 벡터 덧셈과 벡터 곱셈이 정의되어 있다.

  • 임의의 벡터 공간 VVV 는 VVV 에 포함된 모든 벡터 u,v,wu, v, wu,v,w 와 상수 c,dc, dc,d 에 대하여 다음 101010 가지 공리를 만족한다.

    • 덧셈에 대한 아벨군

      • 덧셈에 대하여 닫혀있음 : u+vu+vu+v 로 표기하는 벡터공간 VVV 에 속한 임의의 벡터 u,vu, vu,v 의 합의 벡터 공간 VVV 의 원소이다.

      • 덧셈 결합법칙 : (u+v)+w=u+(v+w)(u + v) + w = u + (v + w)(u+v)+w=u+(v+w)

      • 덧셈 항등원 : u+0=uu+0=uu+0=u 를 만족하는 영벡터가 벡터공간 VVV 에 존재한다.

      • 덧셈 역원 : VVV 에 속하는 모든 벡터 uuu 에 대하여 u+(−u)=0u+(-u)=0u+(−u)=0 을 만족하는 VVV 에 속한 −u-u−u 가 존재한다.

      • 교환법칙 : u+v=v+uu + v = v + uu+v=v+u

    • 곱셈에 대하여 닫혀있음 : VVV 에 속한 모든 벡터 uuu 의 스칼라배인 cucucu 는 VVV 에 속한다.

    • 곱셈 결합법칙 : c(du)=(cd)uc(du) = (cd)uc(du)=(cd)u

    • 벡터 덧셈에 대한 분배법칙 : c(u+v)=cu+cvc(u+v) = cu + cvc(u+v)=cu+cv

    • 스칼라배 덧셈에 대한 분배법칙 : (c+d)u=cu+du(c+d)u = cu + du(c+d)u=cu+du

부분 공간(subspaces) : 다음 333 가지 성질을 만족하는 벡터 공간 VVV 의 부분 집합 HHH 이다.

  1. VVV 의 000벡터는 HHH 에 속한다.

  2. HHH 는 벡터 합에 대해서 닫혀있다.

    • HHH 에 속한 모든 벡터 uuu 와 vvv 에 대해서 그들의 합인 u+vu+vu+v 도 HHH 에 속한다.

  3. HHH 는 스칼라 곱에 대해서 닫혀있다.

    • HHH 에 속한 모든 벡터 uuu 와 스칼라 ccc 에 대해서 벡터 cvcvcv 는 HHH 에 속한다.

  • 이 성질은 벡터 공간 VVV 의 부분공간 HHH 가 그 자체로도 벡터 공간이 됨을 보장해준다.

  • 역으로 모든 벡터공간은 자기 자신의 부분공간이거나 더 큰 벡터공간의 부분공간이다.

  • VVV 의 부분공간이다.
