4.1 벡터 공간과 부분 공간

  • 벡터 공간(vector space) : 벡터의 집합이다.

    • 벡터 공간에는 벡터 덧셈과 벡터 곱셈이 정의되어 있다.

    • 임의의 벡터 공간 VVV 는 VVV 에 포함된 모든 벡터 u,v,wu, v, wu,v,w 와 상수 c,dc, dc,d 에 대하여 다음 101010 가지 공리를 만족한다.

      1. 덧셈에 대하여 닫혀있음 : u+vu+vu+v 로 표기하는 벡터공간 VVV 에 속한 임의의 벡터 u,vu, vu,v 의 합의 벡터 공간 VVV 의 원소이다.

      2. 교환법칙 : u+v=v+uu + v = v + uu+v=v+u

      3. 덧셈 결합법칙 : (u+v)+w=u+(v+w)(u + v) + w = u + (v + w)(u+v)+w=u+(v+w)

      4. 덧셈 항등원 : u+0=uu+0=uu+0=u 를 만족하는 영벡터가 벡터공간 VVV 에 존재한다.

      5. 덧셈 역원 : VVV 에 속하는 모든 벡터 uuu 에 대하여 u+(−u)=0u+(-u)=0u+(−u)=0 을 만족하는 VVV 에 속한 −u-u−u 가 존재한다.

      6. 곱셈에 대하여 닫혀있음 : VVV 에 속한 모든 벡터 uuu 의 스칼라배인 cucucu 는 VVV 에 속한다.

      7. 분배법칙 : c(u+v)=cu+cvc(u+v) = cu + cvc(u+v)=cu+cv

      8. 분배법칙 : (c+d)u=cu+du(c+d)u = cu + du(c+d)u=cu+du

      9. 곱셈 결합법칙 : c(du)=(cd)uc(du) = (cd)uc(du)=(cd)u

      10. 곱셈 항등원 : 1u=u1u = u1u=u

    • 예시

      • n≥1n \geq 1n≥1 에서의 ℜn\Re ^nℜn 이 주요한 벡터 공간의 예시이다. 그래서 우리가 자주 다뤘던 ℜ2,ℜ3\Re ^2, \Re ^3ℜ2,ℜ3 가 벡터 공간이었던 것이다.

  • 부분 공간(subspaces) : 다음 333 가지 성질을 만족하는 벡터 공간 VVV 의 부분 집합 HHH 이다.

    1. VVV 의 000벡터는 HHH 에 속한다.

    2. HHH 는 벡터 합에 대해서 닫혀있다.

      • HHH 에 속한 모든 벡터 uuu 와 vvv 에 대해서 그들의 합인 u+vu+vu+v 도 HHH 에 속한다.

    3. HHH 는 스칼라 곱에 대해서 닫혀있다.

      • HHH 에 속한 모든 벡터 uuu 와 스칼라 ccc 에 대해서 벡터 cvcvcv 는 HHH 에 속한다.

    • 다음 성질은 벡터 공간 VVV 의 부분공간 HHH 가 그 자체로도 벡터 공간이 됨을 보장해준다.

    • 역으로 모든 벡터공간은 자기 자신의 부분공간이거나 더 큰 벡터공간의 부분공간이다.

  • 영 부분 공간(zero subspace) : 벡터 공간 VVV 에 속한 000벡터만으로 이루어진 집합이다.

    • VVV 의 부분공간이다.

    • {0}\{0\}{0} 로 표기한다.

벡터들의 집합에 의해 생성되는 부분 공간

먼저 선형 결합(linear combination) 이란 벡터들에 스칼라를 곱한 후 더한 것이고, Span{v1,…,vp}Span\{v_1, \dots, v_p\}Span{v1​,…,vp​} 는 v1,…,vpv_1, \dots, v_pv1​,…,vp​ 들의 선형 결합으로 표현할 수 있는 벡터들의 집합이라는 것을 떠올리자.

  • Theorem 1 만약 v1,…,vpv_1, \dots, v_pv1​,…,vp​ 들이 벡터 공간 VVV 에 속한 벡터들이면 Span{v1,…,vp}Span\{v_1, \dots, v_p\}Span{v1​,…,vp​} 는 VVV 의 부분공간이다.

4.2 영 공간, 열 공간, 선형 변환

행렬의 영 공간

  • 영 공간(null space) : m×nm \times nm×n 행렬 AAA 의 영 공간 NulANul ANulA 은 동차 방정식 Ax=0Ax=0Ax=0 의 해들의 집합이다.

  • Theorem 2 m×nm \times nm×n 행렬 AAA 의 영 공간은 ℜn\Re ^nℜn 의 부분공간이다. nnn 개의 미지수를 가지는 mmm 개의 방정식인 Ax=0Ax = 0Ax=0 의 모든 해의 집합은 ℜn\Re ^nℜn 의 부분집합이다.

행렬의 열 공간

  • 열 공간(colum space) : m×nm \times nm×n 행렬 AAA 의 열 공간은 AAA 의 열들의 선형 결합으로 만들 수 있는 벡터들의 집합니다.

    • 행렬 AAA 의 열들이 a1,…,ana_1, \dots, a_na1​,…,an​ 일 때 다음과 같이 정의할 수 있다.

      • ColA=Span{a1,…,an}Col A = Span\{a_1, \dots, a_n\}ColA=Span{a1​,…,an​} 이다.

    • AxAxAx 가 행렬 AAA 의 열들의 선형결합이기 때문에 ColACol AColA 에 속하는 벡터를 AxAxAx 로 표현할 수 있고, 이 관점에서 열 공간을 다음과 같이 정의할 수 있다.

  • Theorem 3 m×nm \times nm×n 행렬 AAA 의 열공간은 ℜm\Re ^mℜm 의 부분공간이다.

  • 선형 변환(linear transformation) : 벡터 공간 VVV 에서 벡터 공간 WWW 로의 선형변환 TTT 는 VVV 에 속한 벡터를 WWW 에 속한 벡터 T(x)T(x)T(x) 로 할당하는 규칙으로써 다음 두 성질을 만족해야 한다.

    2. u∈Vu \in Vu∈V 인 모든 uuu 와 모든 스칼라 ccc 에 대하여 T(cu)=cT(u)T(cu)=cT(u)T(cu)=cT(u)

4.3 선형 독립 집합; 기저

  • 선형 독립(linearly independent) : 다음 조건을 만족하는 벡터 공간 VVV 에 속한 벡터들 {v1,…,vp}\{v_1, \dots, v_p\}{v1​,…,vp​} 는 선형 독립이다.

    • 다음 벡터 방정식이 유일한 해로써 자명한 해(c1=0,…,cp=0c_1=0, \dots, c_p=0c1​=0,…,cp​=0) 만을 가진다.

      c1v1+c2v2+⋯+cpvp=0c_1v_1 + c_2v_2 + \dots + c_pv_p = 0c1​v1​+c2​v2​+⋯+cp​vp​=0(1)

  • 선형 종속(linearly dependent) : 방정식 (1) 이 자명한 해 이외의 해를 가지면 벡터집합 [v1,…,vp][v_1, \dots, v_p][v1​,…,vp​] 은 선형 종속이다.
