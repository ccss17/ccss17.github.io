# standalone

설치 권한이 없는 서버에서도 여러가지 CLI 툴들과 zsh 을 사용할 수 있도록 필요한 라이브러리 바이너리와 설정 파일, 라이브러리 후킹을 해주는 스크립트들입니다.

설치 권한이 없음에도 `bat`, `curl`, `fd`, `gotop`, `hyperfine`, `lsd`, `nvtop`, `tmux`, `vim`, `zsh` 을 standalone 하게 사용할 수 있다는 장점이 있지만, 아무래도 standalone 하게 사용하는 것이라서 용량을 좀 차지하게 된다는 단점이 있습니다.
