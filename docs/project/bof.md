# bof 

[bof](https://github.com/ccss17/bof) 는 한동대학교 보안동아리 고스트 신입기수들이 CTF 를 쉽고 재미있게 입문할 수 있도록 제가 제작한 매우 쉬운 Buffer Overflow CTF 들입니다. 동아리 고스트에서는 신입기수에게 2주간 [security-tutorial](https://github.com/ccss17/security-tutorial) 를 통하여 보안 공부를 지도하는데, 첫 1주일간은 리버싱을 배우고 나머지 1주일 동안 이 제가 만든 BOF CTF 를 풀어보게 됩니다. 

## 이 CTF 에 대해서..

일단 BOF 의 개념만 제대로 익히면 다른 취약점을 쉽게 이해할 수 있다고 판단하였고, 1주일동안이라는 시간이 매우 제한적이어서 이렇게 구성하게 되었습니다. 

이 CTF 의 특징은 어떤 취약점을 해킹해보고, 다시 그것을 보완하는 보안 기법을 적용시키고 해킹해보게 합니다. 즉, 다음과 같은 싸이클을 체험하게 합니다.

취약점 해킹해보기 &rarr; (보안기법 추가) &rarr; 취약점 해킹해보기 &rarr; (보안기법 추가) &rarr; ...

이로써 과거로부터 있어왔던 해킹 기법과 취약점이 어떻게 보완되었고, 그것이 다시 어떻게 파훼되었고, 그것이 또 다시 어떻게 보완되었고, 또 어떻게 다시 파훼되었는지 이해하게 합니다. 이는 학생들에게 중요한 통찰력을 얻게 하며, 컴퓨터 보안 공부의 최전방으로 나아가야겠다는 비전을 보여줍니다.

## Docker

한편, 기존에 이 CTF 를 AWS 서버 또는 VM 에서 운영해왔었는데, 여러가지 문제로 인하여 Docker 컨테이너로 완전히 이전시켰습니다. 이로써 Docker 가 주는 많은 이점을 누릴 수 있게 되었죠.
