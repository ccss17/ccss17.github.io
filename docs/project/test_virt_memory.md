
# test_virt_memory

가상 메모리 구성과 물리 메모리의 구성을 이해하기 위하여 시작된 개인적인 프로젝트입니다. 

이 프로젝트의 특징은 가상 메모리 시스템 구조를 C 언어 프로그램을 통하여 알아봄과 동시에 그것이 어떤 물리 메모리에 대응되는지 알아본다는 것입니다. 

저는 가상 메모리 주소값이 어떤 물리 메모리에 대응되는지 알고 싶었습니다만, 방방곡곡 알아보아도 그것은 불가능하다는 답변밖에 없었지요. 하지만 [Meltdown](https://ccss17.github.io/paper/meltdown/) 을 공부하던 도중 이것의 POC 코드를 조금만 응용하면 가상 메모리 주소값에 대응되는 물리 메모리 주소값을 알아낼 수 있다는 것을 알게 되었습니다.

또한 가상 메모리 주소에 있는 특정 값을 외부 프로세스가 어떻게 조작할 수 있는지 실험하는 내용도 있습니다.
