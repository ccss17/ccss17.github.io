
# ubuntu-unminimize

[ProgrammerBase](../../ProgrammerBase/readme/) 를 리눅스 환경에서 실습하기 위하여 제작한 Docker 이미지입니다.

일반 Ubuntnu Linux 의 Docker 이미지는 minimize 가 되어있습니다. 즉, 실제 Ubuntu 리눅스 환경을 사용하기에 어려움이 있죠. 그래서 그것을 unminimize 하고 리눅스를 실습할 수 있는 여타 환경을 설정해둔 컨테이너가 필요해서 시작된 프로젝트입니다.
