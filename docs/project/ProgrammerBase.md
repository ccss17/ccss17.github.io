# ProgramerBase

컴퓨터 공학에 입문한 신입생을 위한 1주일 간의 가이드입니다.

여러가지 프로그래머로써 꼭 알아야 할 지식을 알려줍니다. 

이 가이드의 특징은 여러가지 프로그래머로써 알아야할 지식들을 알려줄 뿐만 아니라 컴퓨터가 탄생하게 된 수학적인 배경을 설명하고, 앞으로 컴퓨터가 어떻게 인간의 지능을 초월할 수 있는지에 대한 기술적 특이점에 대한 내용도 설명함으로써 컴퓨터에 대한 통찰력을 학생에게 부여해줍니다.
