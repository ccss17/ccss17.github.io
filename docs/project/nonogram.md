# nonogram

이 프로그램은 네모로직 수학퍼즐을 풀던 중 "이거 어차피 일련의 규칙대로 푸는건대 왜 내가 풀어야하지? 기계가 풀도록 하면 되잖아" 라는 생각이 들어서 만든 프로그램입니다.

그냥 코드가 생각나는대로 대충 대충 만든 프로그램이라서 Time Complexity 같은 건 고려하지 않았습니다. 너무 대충 만들어서 아마 Time Complexity 가 엄청 높을껄요?..
