# 역사의 정의

- 존재론적 역사 : 인간에게 과거에 무슨 일이 일어났는가.

    - 사건으로의 역사이다.

- 인식론적 역사 : 과거에 일어난 것들의 기록.

    - 기록의로의 역사이다.

# 어원

- 그리스 : inquiry

- 독일 : Geschichte

    - 어떤 것이 일어났다는 뜻 

- 중국 : 歷史

# 역사적 사실

- 역사 속 사실 : 역사는 팩트을 포함하기에 팩트이 역사를 이루는 핵심이다.

- 역사적 팩트에 대한 직면 : 역사적 팩트가 항상 과거에 일어났던 것은 아니다.

## 역사적 사실의 재구성

- 팩트의 의미 : 어떤 것의 있는 그대로의 것.

    - 존재론적 의미 : 사건 그대로의 것.

    - 인식론적 의미 : 사건의 기록.

- 역사적 사실은 어느 때에나 관차될 수 있는 것이 아니다.

    - 역사적 사실은 역사적 재료에 의하여 재구성되어야 한다.

    - 역사가들이 이러한 재구성 작업을 한다.

- 역사적 사실의 재구성

    - 역사적 사료

    - 역사적 방법론

    - 역사가의 역할

# 역사적 방법론

- 방법론의 중요성

    - 과학과 철학의 차이

        - 과학: 자연

        - 인간성: 인간존재 

    - 과학적 방법론이 가장 믿을만 한다.

        - 관찰, 가설, 실험, 추상화
