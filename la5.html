고윳값과 고유 벡터

고유벡터(eigenvector) : n×nn \times nn×n 행렬 AAA 의 고유 벡터는 어떤 스칼라 λ\lambdaλ 에 대하여

Ax=λx Ax = \lambda x Ax=λx

를 만족하는 영이 아닌 벡터 xxx 이다.

  • 고유벡터의 정의 Ax=λxAx = \lambda xAx=λx 를 Ax=λExAx = \lambda E xAx=λEx 로 쓰면

    (A−λE)x=0 (A- \lambda E) x = 0 (A−λE)x=0

    을 얻는다. 여기에서 (A−λE)(A- \lambda E)(A−λE) 의 역행렬이 존재하면 양변에 곱하여

    (A−λE)−1(A−λE)x=(A−λE)−10 (A- \lambda E) ^{-1}(A- \lambda E) x = (A- \lambda E)^{-1} 0 (A−λE)−1(A−λE)x=(A−λE)−10
    x=(A−λE)−10=0 x = (A- \lambda E)^{-1} 0 = 0 x=(A−λE)−10=0

    이 된다. 그러나 x=0x = 0x=0 은 xxx 가 영이 아닌 벡터라는 정의의 조건에 모순되므로 (A−λE)(A- \lambda E)(A−λE) 가 역행렬을 갖는다는 가정이 틀렸다는 것을 알 수 있다.

    따라서 고유벡터가 존재하려면

    det⁡(A−λE)=0 \det (A - \lambda E) = 0 det(A−λE)=0

    이어야 한다.

  • 예시

    A=[1652],u=[6−5],v=[3−2] A = \begin{bmatrix} 1&6\\5&2 \end{bmatrix}, u = \begin{bmatrix} 6\\-5 \end{bmatrix}, v = \begin{bmatrix} 3\\-2 \end{bmatrix} A=[15​62​],u=[6−5​],v=[3−2​]

    에서 u,vu, vu,v 가 AAA 의 고유 벡터인지 확인하자.

    Au=[1652][6−5]=[−2420]=−4[6−5]=−4u Au = \begin{bmatrix} 1&6\\5&2 \end{bmatrix}\begin{bmatrix} 6\\-5 \end{bmatrix} = \begin{bmatrix} -24\\20 \end{bmatrix} = -4 \begin{bmatrix} 6\\-5 \end{bmatrix} = -4u Au=[15​62​][6−5​]=[−2420​]=−4[6−5​]=−4u
    Av=[1652][3−2]=[−911]≠λ[3−2] Av = \begin{bmatrix} 1&6\\5&2 \end{bmatrix}\begin{bmatrix} 3\\-2 \end{bmatrix} = \begin{bmatrix} -9\\11 \end{bmatrix} \neq \lambda \begin{bmatrix} 3\\-2 \end{bmatrix} Av=[15​62​][3−2​]=[−911​]​=λ[3−2​]

    따라서 uuu 는 −4-4−4 에 대응하는 고유벡터이지만

    vvv 는 AvAvAv 가 vvv 의 스칼라배가 아니므로 고유 벡터가 아니다.

고윳값(eigenvalue) : n×nn \times nn×n 행렬 AAA 에서

Ax=λx Ax = \lambda x Ax=λx

의 영이 아닌 해, 즉 자명하지 않은(nontrivial) 해인 벡터 xxx 가 존재할 때 스칼라 λ\lambdaλ 를 고윳값이라 한다.

  • 예시

    A=[1652] A = \begin{bmatrix} 1&6\\5&2 \end{bmatrix}A=[15​62​]

    의 고윳값이 777 임을 보이자.

    이것은 다음 등식이 영이 아닌 해를 가지는 것과 동치이다.

    Ax=7x Ax = 7x Ax=7x

    이 등식은

    (A−7I)x=0 (A - 7I)x = 0 (A−7I)x=0

    과 같으므로

    A−7I=[1652]−[7007]=[−665−5] A - 7I = \begin{bmatrix} 1&6\\5&2 \end{bmatrix} - \begin{bmatrix} 7&0\\0&7 \end{bmatrix} = \begin{bmatrix} -6&6\\5&-5 \end{bmatrix} A−7I=[15​62​]−[70​07​]=[−65​6−5​]

    이다. A−7IA - 7IA−7I 의 열들은 선형종속이므로 영이 아닌 해를 가진다.

    그러므로 777 은 AAA 의 고윳값이다.

인공에서 선생님 없는 학습이라 불리는 비지도학습에서 주성분 분석이라는 기법이 사용되는데, 이것으 다차원 데이터를 다루기 쉽게 222 차원이나 333 차원으로 압축하는 방법이다. 데이터가 많이 흩어져 있는 분포 상황에서 문제를 해결하기 위해 고유값과 고유벡터가 사용된다.

또 고유값은 어떤 데이터가 특징을 얼마나 잘 설명할 수 있는지 가늠할 때도 사용된다. 기여율이라는 것은 각 주성분, 즉 고유벡터에 대응되는 고유값을 전체 고유값들의 총합을 나눈것으로 주성분이 데이터를 얼마나 잘 설명할 수 있는지 평가할 때 사용된다.

고유벡터와 고유값의 기하학적 의미 : 어떤 행렬에 벡터를 곱하는 선형변환 연산을 했을 때 일반적으로 벡터의 방향이 바뀌지만

고유벡터로 선형변환을 하면 방향이 바뀌지 않거나 역방향이 된다.

  • 따라서 고유값은 선형변환 전과 후의 벡터 길이의 비율로 볼 수 있다.

  • 다음 그래프를 보자.

    벡터 uuu 를 벡터 AC→,AF→\overrightarrow{AC}, \overrightarrow{AF}AC,AF 로 변환시킨 선형변환은 고유벡터가 사용된 것이고, 고유값이 길이의 비율에 해당한다.

    벡터 www 를 벡터 AE→\overrightarrow{AE}AE 로 선형변환시킨 것도 방향이 바뀌지 않았으므로 고유벡터가 사용된 것이다.

    그러나 벡터 u,wu, wu,w 가 벡터 ccc 로 선형변환되었다면 방향이 바뀌었으므로 고유벡터가 사용되지 않은 것이다.

고유공간(eigenspace) : 행렬 AAA 를 고윳값에 대하여 정리한 행렬방정식

(A−λI)x=0 (A - \lambda I)x = 0 (A−λI)x=0

고유방정식(eigenvalue equation) : 행렬 AAA 에 대한 λ\lambdaλ 의 방정식

det⁡(A−λE)=0 \det (A - \lambda E) = 0 det(A−λE)=0
