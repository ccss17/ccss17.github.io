확률

  • 확률(probability) : 모든 경우의 수(표본공간)에 대하여 특정한 사건이 발생하는 경우의 수의 비율이다.

    • 다시 말해 전사건(표본공간)의 경우의 수에 대한 어떤 사건이 일어날 경우의 수에 대한 비율이다.

    • 사건 AAA 에 대한 확률을

      P(A) P(A) P(A)

      로 나타낸다.

    • 모든 경우의 수가 nnn 이고 사건 AAA 가 일어나는 경우의 수가 aaa 라면 사건 AAA 가 일어날 확률 ppp 는 다음과 같다.

      P(A)=p=an P(A) = p = \frac{a}{n} P(A)=p=na​

    • 확률 ppp 는 0≤p≤10 \leq p \leq 10≤p≤1 이다.

    • 사건 AAA 가 일어나지 않을 확률은 1−p1 - p1−p 이다.

    • 예시

      동전을 던질 때 앞면이 나오는 가능성은 앞면, 뒷면 222 가지 경우 중 111 가지이므로 12\dfrac{1}{2}21​ 이다.

  • 사건의 경우의 수 : 사건 AAA 에 대한 경우의 수를 n(A)n(A)n(A) 로 정의한다.

    • 전사건의 경우의 수, 즉 표본공간 SSS 의 원소를

      n(S) n(S) n(S)

      로 나타낸다.

  • 수학적 확률(mathematical probability) : 표본공간 SSS 안에서 각각의 근원사건이 일어날 가능성이 같을 때

    사건 AAA 가 일어날 수학적 확률은

    P(A)=n(A)n(S) \boxed{P(A) = \frac{n(A)}{n(S)}} P(A)=n(S)n(A)​​

    이다.

    • 예시

      주사위를 던졌을 때 666 이 나올 수학적 확률을 구해보자.

      주사위를 던졌을 때 각각의 눈(1,2,3,4,5,61,2,3,4,5,61,2,3,4,5,6)이 나올 경우의 수는 666 이고

      그 중에서 666 이 나오는 경우의 수는 111 이므로 때 666 이 나올 수학적 확률은

      ∴16\therefore \dfrac{1}{6}∴61​

      이다. ■

    • 예시

      흰 공, 검은 공, 파란 공이 222 개씩 들어있는 주머니에서 공 111 개를 꺼낼 때

      파란 공이 111 개 나올 확률을 구해보자.

      666 개의 공에서 111 개를 꺼내는 경우의 수는 6C1=6{}_{6}C_{1} = 66​C1​=6 이고

      파란 공 111 개를 꺼내는 경우의 수는 2C1=2{}_{2}C_{1} = 22​C1​=2 이므로 구하고자 하는 확률은

      ∴26=13 \therefore \frac{2}{6} = \frac{1}{3} ∴62​=31​

      이다. ■

    • 예시

      흰 공, 검은 공, 파란 공이 222 개씩 들어있는 주머니에서 공 333 개를 꺼낼 때

      흰 공, 검은 공, 파란 공이 한 개씩 나올 확률을 구해보자.

      666 개의 공에서 333 개의 공을 꺼내는 경우의 수는 6C3=20{}_{6}C_{3} = 206​C3​=20 이고

      흰 공, 검은 공, 파란 공이 한 개씩 나오는 경우의 수는

      2C1×2C1×2C1=8 {}_{2}C_{1} \times {}_{2}C_{1} \times {}_{2}C_{1} = 8 2​C1​×2​C1​×2​C1​=8

      이므로 구하고자 하는 확률은

      ∴820=25 \therefore \frac{8}{20} = \frac{2}{5} ∴208​=52​

      이다. ■

  • 통계적 확률 : 같은 시행을 nnn 번 반복하여 사건 AAA 가 일어난 횟수를 rnr _{n}rn​ 이라 할 때

    시행 횟수 nnn 을 한없이 크게 함에 따라 상대도수

    rnn \boxed{\frac{r_n}{n}} nrn​​​

    이 다가가는 일정한 값 ppp 를 사건 AAA 가 일어날 통계적 확률이라 한다.

    • 즉 통계적 확률은

      P(A)=lim⁡n→∞rnn=p P(A) = \lim_{n \to \infty} \frac{r_n}{n} = p P(A)=n→∞lim​nrn​​=p

      이고 이 극한의 수렴값은 수학적 확률과 같아진다.

    • 실제 자연 현상과 사회 현상에서는 근원사건이 일어날 가능성이 매번 똑같지 않다. 따라서 실제로는 여러 번의 시행을 반복하여 얻어지는 상대도수를 통하여 근원사건의 가능성을 근사적으로 추정한다.

    • 예시

      농구 선수가 자유투를 성공할 확률을 구한다고 하자.

      농구 선수가 자유투를 성공시킬 근원사건의 확률은 매번 동일하지 않다. 그때그때의 컨디션과 상황과 조건과 환경에 영향을 받기 때문이다.

      자유투를 성공한 횟수를 rnr_nrn​, 자유투를 던진 횟수를 nnn 이라고 할 때 이 경우 충분한 시행 횟수를 반복하여

      rnn \frac{r_n}{n} nrn​​

      으로 가능성을 계산한다.

      이때 nnn 이 크면 클수록 가능성은 정확한 수치를 갖게 될 것이다.

      lim⁡n→∞rnn \lim_{n \to \infty} \frac{r_n}{n} n→∞lim​nrn​​

      즉 위와 같은 수식이 정확한 가능성을 보여준다. 그러나 실제 현실에서는 농구 선수에게 무한번 자유투를 던지게 할 수 없는 노릇이니 그냥 충분한 횟수를 던지게 해야 한다.

  • 큰 수의 법칙(law of large number) : 통계적 확률에서 시행 횟수 nnn 이 한없이 커질 때 그 극한이 수학적 확률과 같아진다.

    • 예시

      동전을 던져 앞면이 나오는 통계적 확률을 구한다고 하자.

      시행 101010 을 했더니 수학적 확률이 210\dfrac{2}{10}102​ 이 나왔고

      시행 202020 을 했더니 수학적 확률이 920\dfrac{9}{20}209​ 이 나왔다고 하자.

      실제로 수학적 확률은 12\dfrac{1}{2}21​ 이고 시행을 무한히 반복하면 통계적 확률이 이 12\dfrac{1}{2}21​ 에 가까워진다.

  • 기하학적 확률 : 영역 SSS 안에서 각각의 점을 잡을 가능성이 같고 sum(A)\text{sum} (A)sum(A) 을 영역 AAA 의 넓이라고 할 떄

    영역 SSS 에서 임의로 잡은 점이 영역 AAA 에 속할 기하학적 확률은

    P(A)=sum(A)sum(S) \boxed{P(A) = \frac{\text{sum}(A)}{\text{sum}(S)}} P(A)=sum(S)sum(A)​​

    이다.

  • 확률의 성질 : 전사건 SSS 과 임의의 사건 AAA 에 대하여 다음이 성립한다.

    1. 0≤P(A)≤1\boxed{ 0 \leq P(A) \leq 1}0≤P(A)≤1​

    2. P(S)=1\boxed{ P(S) = 1}P(S)=1​

    3. P(∅)=0\boxed{ P(\emptyset ) = 0}P(∅)=0​

    • 증명

      어떤 시행에서 일어나는 모든 경우의 수를 nnn 이라 하고

      특정 사건 AAA 가 일어나는 경우의 수를 rrr 이라 하자.

      그러면 자명하게

      0≤r≤n 0 \leq r \leq n 0≤r≤n

      이 성립한다. 이때 양변을 nnn 으로 나누어

      0≤rn≤1 0 \leq \frac{r}{n} \leq 1 0≤nr​≤1

      를 얻는데 rn\dfrac{r}{n}nr​ 은 수학적 확률 P(A)P(A)P(A) 이므로

      ∴0≤P(A)≤1 \therefore 0 \leq P(A) \leq 1 ∴0≤P(A)≤1

      를 얻는다. ■

      이때 전사건 SSS 가 일어날 경우의 수는 nnn 이므로

      ∴P(S)=nn=1 \therefore P(S) = \frac{n}{n} = 1 ∴P(S)=nn​=1

      이다. ■

      또 공사건 ∅\emptyset∅ 가 일어날 경우의 수는 000 이므로

      ∴P(∅)=0n=0 \therefore P(\emptyset ) = \frac{0}{n} = 0 ∴P(∅)=n0​=0

      이다. ■

    • 예시

      1,3,5,7,9,111, 3, 5, 7, 9, 111,3,5,7,9,11 이 적혀있는 주사위를 던져 짝수가 나올 확률을 구해보자.

      주사위에 있는 눈이 모두 홀수이므로 짝수가 나올 경우의 수는 000 이고 이것은 공사건이다. 그러므로 짝수가 나올 확률은

      ∴06=0 \therefore \dfrac{0}{6} = 0 ∴60​=0

      이다. ■

      그러면 홀수가 나올 확률을 구해보자.

      주사위에 있는 눈이 모두 홀수이므로 홀수가 나올 경우의 수는 666 이고 이것은 전사건이다. 그러므로 홀수가 나올 확률은

      ∴66=1 \therefore \dfrac{6}{6} = 1 ∴66​=1

      이다. ■

확률의 덧셈정리

  • 확률의 덧셈 : 두 사건 AAA 와 BBB 가 동시에 일어나지 않을 때, 사건 AAA 와 BBB 가 발생할 확률을 각각 p,qp, qp,q 라 하면 사건 AAA 또는 BBB 가 발생할 확률은 p+qp + qp+q 이다.

  • 확률의 곱 : 두 사건 AAA 와 BBB 가 서로 영향을 미치지 않을 때, 사건 AAA 와 BBB 가 발생할 확률을 각각 p,qp, qp,q 라 하면 사건 AAA 또는 BBB 가 동시에 발생할 확률은 p×qp \times qp×q 이다.

  • 확률의 덧셈정리(addition theorem of probability) : 표본공간 SSS 의 부분집합인 두 사건 A,BA, BA,B 에 대하여

    P(A∪B)=P(A)+P(B)−P(A∩B) P(A \cup B) = P(A)+P(B)-P(A \cap B) P(A∪B)=P(A)+P(B)−P(A∩B)

    이다.

    • 증명

      두 사건 집합 A,BA, BA,B 에 대하여 합집합의 원소의 개수

      n(A∪B)=n(A)+n(B)−n(A∩B) n(A \cup B) = n(A) + n(B) - n(A \cap B) n(A∪B)=n(A)+n(B)−n(A∩B)

      가 성립함을 이미 알고 있다.

      여기에서 전사건 n(S)n(S)n(S) 을 나누면

      n(A∪B)n(S)=n(A)n(S)+n(B)n(S)−n(A∩B)n(S) \dfrac{n(A \cup B)}{n(S)} = \dfrac{n(A)}{n(S)} + \dfrac{n(B)}{n(S)} - \dfrac{n(A \cap B)}{n(S)} n(S)n(A∪B)​=n(S)n(A)​+n(S)n(B)​−n(S)n(A∩B)​

      을 얻는데 이것은 곧 확률이므로

      ∴P(A∪B)=P(A)+P(B)−P(A∩B) \therefore P(A \cup B) = P(A) + P(B) - P(A \cap B) ∴P(A∪B)=P(A)+P(B)−P(A∩B)

      을 얻는다. ■

  • 배반사건의 확률의 덧셈정리 : 두 사건 A,BA, BA,B 가 배반사건이면, 즉 A∩B=∅A \cap B = \emptysetA∩B=∅ 이면

    P(A∪B)=P(A)+P(B) \boxed{P(A \cup B) = P(A)+P(B) } P(A∪B)=P(A)+P(B)​

    이다.

    • 증명

      두 사건 A,BA, BA,B 가 배반사건이면 A∩B=∅A \cap B = \emptysetA∩B=∅ 이므로

      P(A∩B)=P(∅)=0 P(A \cap B) = P(\emptyset ) = 0 P(A∩B)=P(∅)=0

      이 되어 확률의 덧셈정리

      P(A∪B)=P(A)+P(B)−P(A∩B) P(A \cup B) = P(A)+P(B)-P(A \cap B) P(A∪B)=P(A)+P(B)−P(A∩B)

      에서

      ∴P(A∪B)=P(A)+P(B) \therefore P(A \cup B) = P(A)+P(B) ∴P(A∪B)=P(A)+P(B)

      이다. ■

    • 또한 일반적으로 nnn 개의 사건 A1,A2,…,AnA_1, A_2, \dots, A_nA1​,A2​,…,An​ 이 서로 배반이면

      P(A1∪A2∪⋯∪An)=P(A1)+P(A2)+⋯+P(An) \boxed{P(A_1 \cup A_2 \cup \dots \cup A_n) = P(A_1)+P(A_2)+\dots+P(A_n) } P(A1​∪A2​∪⋯∪An​)=P(A1​)+P(A2​)+⋯+P(An​)​

      이다.

  • 여사건의 확률 : 표본공간 SSS 의 부분집합인 사건 AAA 의 여사건 ACA ^{C}AC 가 일어날 확률은

    P(AC)=1−P(A) \boxed{P(A ^{C}) = 1 - P(A)} P(AC)=1−P(A)​

    이다.

    • 증명

      사건 AAA 와 사건 AAA 가 일어나지 않는 여사건 ACA ^{C}AC 은 서로 배반이므로

      P(A∪AC)=P(A)+P(AC) P(A \cup A ^{C}) = P(A)+P(A ^{C}) P(A∪AC)=P(A)+P(AC)

      이다. 이때 표본공간 SSS (전사건) 에 대하여 A∪AC=SA \cup A ^{C} = SA∪AC=S 이므로

      P(A∪AC)=P(S)=1 P(A \cup A ^{C}) = P(S) = 1 P(A∪AC)=P(S)=1

      이고 여기에서

      P(A∪AC)=P(S)=P(A)+P(AC)=1 P(A \cup A ^{C}) = P(S) = P(A)+P(A ^{C}) = 1 P(A∪AC)=P(S)=P(A)+P(AC)=1

      이다.

      그러므로

      P(A)+P(AC)=1 P(A)+P(A ^{C}) = 1 P(A)+P(AC)=1

      를 얻고 최종적으로

      ∴P(AC)=1−P(A) \therefore P(A ^{C}) = 1-P(A)∴P(AC)=1−P(A)

      이다. ■

    • 여사건은 구하고자 하는 사건의 확률을 구하기 어려울 때 이용된다.

    • 예시

      남자 333 명과 여자 444 명 중 회장 222 명을 선출하려 할 때 남자가 적어도 111 명 선출될 확률을 구해보자.

      이 확률은 남자가 선출되지 않는 사건 AAA 의 여사건 ACA ^{C}AC 이다.

      따라서 사건 AAA 가 일어날 확률은 777 중에서 222 명을 뽑는 경우의 수에 대한 여자 444 명 중 222 명을 뽑는 경우의 수의 비

      P(A)=4C27C2=27 P(A) = \frac{{}_{4}C_{2}}{{}_{7}C_{2}} = \dfrac{2}{7} P(A)=7​C2​4​C2​​=72​

      이고, 사건 AAA 의 여사건이 일어날 확률은

      ∴P(AC)=1−27=57 \therefore P(A ^{C}) = 1 - \dfrac{2}{7} = \dfrac{5}{7} ∴P(AC)=1−72​=75​

      이다. 이것이 남자가 적어도 111 명 선출되는 확률이다. ■

조건부 확률

  • 조건부확률(conditional probability) : 확률이 000 이 아닌 사건 AAA 가 일어났다고 가정할 때

    사건 BBB 가 일어날 확률

    P(B∣A)=n(A∩B)n(A)=P(A∩B)P(A)=P(A∩B)P(A∩B)+P(A∩BC) \boxed{P(B|A) = \frac{n(A \cap B)}{n(A)} = \frac{P(A \cap B)}{P(A)}= \dfrac{P(A \cap B)}{P(A \cap B) + P(A \cap B ^{C})}} P(B∣A)=n(A)n(A∩B)​=P(A)P(A∩B)​=P(A∩B)+P(A∩BC)P(A∩B)​​

    을 사건 AAA 가 일어났을 때의 사건 BBB 의 조건부확률이라 한다.

    • 증명

      확률이란 표본공간 SSS 의 원소의 개수, 즉 전사건의 경우의 수에 대한 사건의 경우의 수의 비율이었음을 기억하자.

      조건부 확률이란 사건 AAA 가 일어났다는 가정하에 사건 BBB 가 일어난 확률이므로 새로운 표본공간 AAA 을 설정하고 A∩BA \cap BA∩B 가 일어날 확률을 구하는 것이다.

      따라서 조건부 확률은 사건 AAA 가 일어날 경우의 수 n(A)n(A)n(A) 을 새로운 전사건의 경우의 수로 두고, 사건 BBB 가 일어날 경우의 수의 비를 구하는 것이다.

      그런데 이때 사건 AAA 가 반드시 일어나야 하기 때문에 사건 AAA 에 속하지 않는 사건은 확률에 포함시켜서는 안된다. 따라서 위 그림처럼 BBB 가 일어날 확률이 아니라 A∩BA \cap BA∩B 가 일어날 확률을 구해야 한다.

      그러므로 결국 조건부 확률은 P(B∣A)=n(A∩B)n(A)P(B|A) = \dfrac{n(A \cap B)}{n(A)}P(B∣A)=n(A)n(A∩B)​ 이다. 그런데 이때

      P(A)=n(A)n(S),P(A∩B)=n(A∩B)n(S) P(A) = \dfrac{n(A)}{n(S)}, P(A \cap B) = \dfrac{n(A \cap B)}{n(S)} P(A)=n(S)n(A)​,P(A∩B)=n(S)n(A∩B)​

      이므로

      ∴P(B∣A)=n(A∩B)n(A)=n(A∩B)n(S)n(A)n(S)=P(A∩B)P(A) \therefore P(B|A) = \dfrac{n(A \cap B)}{n(A)} = \dfrac{\dfrac{n(A \cap B)}{n(S)}}{\dfrac{n(A)}{n(S)}} = \dfrac{P(A \cap B)}{P(A)} ∴P(B∣A)=n(A)n(A∩B)​=n(S)n(A)​n(S)n(A∩B)​​=P(A)P(A∩B)​

      이다. ■

사건의 독립과 종속

  • 독립사건(independent) : 두 사건 AAA 와 BBB 에서 한 사건의 결과가 다른 사건에 영향을 주지 않을 때, 즉

    P(B∣A)=P(B) P(B|A) = P(B) P(B∣A)=P(B)

    또는

    P(A∣B)=P(A) P(A|B) = P(A) P(A∣B)=P(A)

    일 때 AAA 와 BBB 를 독립사건이라 한다.

    • 다음 등식이 성립하면 두 사건 A,BA, BA,B 는 서로 독립이다.

      P(B∣A)=P(B) P(B|A) = P(B) P(B∣A)=P(B)

      또는

      P(A∣B)=P(A) P(A|B) = P(A) P(A∣B)=P(A)

      또는

      P(B∣AC)=P(B) P(B|A ^{C}) = P(B) P(B∣AC)=P(B)

      또는

      P(A∣BC)=P(A) P(A|B ^{C}) = P(A) P(A∣BC)=P(A)

    • P(A)≠0,P(B)≠0P(A) \neq 0, P(B) \neq 0P(A)​=0,P(B)​=0 인 두 사건 A,BA, BA,B 가 독립인 것과

      P(A∩B)=P(A)P(B) P(A \cap B) = P(A)P(B) P(A∩B)=P(A)P(B)

      인 것은 동치이다. 그러므로

      P(B∣A)=P(B)  ⟺  P(A∩B)=P(A)P(B) P(B|A) = P(B) \iff P(A \cap B) = P(A)P(B) P(B∣A)=P(B)⟺P(A∩B)=P(A)P(B)

      이다.

    • 예시

      주머니에 공이 101010 개가 있고 그 중에서 흰 공이 333 개 있다고 하자. 이때 두 사람이 공을 뽑았을 때 모두 흰공이 나올 확률을 계산해보자.

      첫번째 사람이 공을 뽑는 사건을 AAA, 두번째 사람이 공을 뽑는 사건을 BBB 라고 하자.

      그런데 두 사람이 흰 공을 뽑으려고 한다면 이것은 첫번째 사람이 공을 뽑고 다시 주머니에 공을 넣는 경우와 공을 넣지 않는 경우를 생각할 수 있다.

      만약 첫번째 사람이 공을 뽑고 다시 주머니에 넣으면 공의 개수에 변화가 없으므로 확률 계산에 사용되는 경우의 수도 변함이 없다. 따라서 두번째 사람이 공을 뽑을 때에도 영향을 미치지 않으므로 사건 A,BA, BA,B 는 독립사건이다. 즉,

      P(A)=310,P(B∣A)=310 P(A) = \dfrac{3}{10}, P(B|A) = \dfrac{3}{10} P(A)=103​,P(B∣A)=103​

      이므로 구하고자 하는 확률 P(A∩B)P(A \cap B)P(A∩B) 는

      ∴P(A∩B)=P(A)P(B∣A)=310×310=9100 \therefore P(A \cap B) = P(A)P(B|A) = \dfrac{3}{10} \times \dfrac{3}{10} = \dfrac{9}{100} ∴P(A∩B)=P(A)P(B∣A)=103​×103​=1009​

      이다. ■

  • 종속사건(dependent) : 두 사건 AAA 와 BBB 에서 한 사건의 결과가 다른 사건에 영향을 줄 때, 즉

    P(B)≠P(B∣A),P(A)≠P(A∣B) P(B) \neq P(B|A), P(A) \neq P(A|B) P(B)​=P(B∣A),P(A)​=P(A∣B)

    일 때 AAA 와 BBB 를 종속사건이라 한다.
