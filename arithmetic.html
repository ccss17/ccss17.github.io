양 : 현상, 물체, 또는 물질의 크기이다.

  • 분리량과 연속량으로 나뉘는데 전자에는 물체의 개수 등이 포함되고 후자는 길이나 넓이 등이 포함된다.

등호(===) : 수의 같음을 나타낸다.

뺄셈(−-−) : 전항의 수에 후항의 수만큼의 양을 제거한 양을 나타내는 이항연산이다.

비(비율) : 어떤 양을 기준으로 다른 양의 크기의 정도를 나타내는 값이다.

  • 두 수 aaa 와 bbb 의 비는 a:ba : ba:b 로 나타내고 값은 ab\frac{a}{b}ba​ 이다.

수의 연산

약수 : 어떤 수를 나누어떨어지게 하는 수이다.

배수 : 어떤 수에 정수를 곱한 수이다.

공약수 : 두 수의 공통인 약수이다.

최대공약수 : 공약수 중 가장 큰 수이다.

공배수 : 두 수의 공통인 배수이다.

소수(decimal) : 각각의 자리에 놓인 숫자와 소수점을 통해 나타낸 실수이다.

  • 음이 아닌 실수 rrr 의 소수 표기를

    r=r0.r1r2… r = r_0.r_1r_2 \dots r=r0​.r1​r2​…

    라고 한다.

  • 소수점 왼쪽에 놓인 숫자들을 실수의 정수 부분, 소수점 오른쪽에 놓인 숫자들을 실수의 소수 부분이라 한다.

  • 예시

    1/2=0.5 1/2 = 0.5 1/2=0.5
    1/3=0.33333… 1/3 = 0.33333 \dots 1/3=0.33333…

유한소수(finite decimal) : 소수점 아랫자리가 유한한 수를 갖는 것을 유한소수라 한다.

  • 예시

    1/2=0.5 1/2 = 0.5 1/2=0.5

유한소수(infinite decimal) : 소수점 아랫자리가 무한한 수를 갖는 것을 무한소수라 한다.

  • 예시

    1/3=0.33333… 1/3 = 0.33333 \dots 1/3=0.33333…

순환소수 : 소수점 아래 어떤 자리에서부터 일정한 숫자의 배열이 계속해서 되풀이 되는 무한소수이다.

  • 예시

    0.153153153… 0.153153153 \dots 0.153153153…

    는 다음과 같이 나타낸다.

    0.153‾ 0.\overline{153} 0.153

    또는 다음과 같이 점으로 나타낼 수도 있다.

    0.1˙53˙ 0.\dot{1}5 \dot{3} 0.1˙53˙

최소공배수 : 공배수 중 가장 작은 수이다.

소수(prime number) : 111 보다 큰 자연수 중 111 과 자기 자신만을 약수로 가지는 수이다.

  • P\mathbb{P}P 로 표기된다.

서로소 : 최대공약수가 111 인 두 수의 관계이다.

역연산 :

항등원 : 임의의 수 aaa 에 대하여 어떤 수를 연산했을 때 처음의 수 aaa 가 되도록 만들어주는 수이다.

  • 군론을 비롯한 대수학에서 정의된다.

  • 항등원은 수가 속한 집합과 이항연산의 종류에 따라 달라진다.

  • 자연적 의미로써는 어떤 양을 다른 양과 같게 만드는 수학적 관계이다.

  • 예시

    집합 연산자 항등원
    실수 +,−+, -+,− 000
    실수 ×\times× 111
    정사각행렬 행렬 덧셈 영행렬
    정사각행렬 행렬 곱셈 단위행렬
    함수 합성함수 항등함수

역원(inverse element) : 두 원소를 연산한 결과가 항등원일 때, 한 원소에 대하여 다른 원소를 이르는 말이다.

  • 덧셈에서의 반수와 곱셈에서의 역수를 일반화한 개념이다.

  • 예시

    • 덧셈의 역원은 부호가 반대인 수(반수)이다. a+(−a)=0a + (-a) = 0a+(−a)=0

    • 곱셈의 역원은 지수의 부호가 반대인 수(역수)이다. a⋅(a−1)=1a \cdot (a^{-1}) = 1a⋅(a−1)=1

수의 분류

페아노 공리계

자연수(natural number) : 1,2,3,...1, 2, 3, ...1,2,3,... 등의 수의 체계.

  • N\NN 로 표기한다.

  • 페아노 공리계와 집합론적 정의로 엄밀히 정의된다.

정수(integer) : 음의 정수(−1,−2,−3,...-1, -2, -3, ...−1,−2,−3,...) 와 000, 그리고 양의 정수(1,2,3,...1, 2, 3, ...1,2,3,...)$ 로 이루어진 수의 체계.

  • Z\mathbb{Z}Z 로 표기한다.

  • 대수학에서 정수환으로 엄밀히 정의된다.

유리수(rational number) : 두 정수의 비율로 나타낼 수 있는 수. 단, 분모가 000 이 아니어야 한다.

  • 분모가 111 일 수 있으므로 모든 정수는 유리수이다.

  • Q\mathbb{Q}Q 로 표기한다.

  • 대수학에서 정수환의 분수체로 나타낼 수 있는 유리수체로 엄밀히 정의된다.

무리수(irrational number) : 두 정수의 비의 형태로 나타낼 수 없는 실수를 뜻한다.

  • 분수로 나타낼 수 없는 소수라고도 할 수 있다.

  • I\mathbb{I}I 로 표기한다.

실수(real number) : 유리수 집합과 무리수 집합을 합한 수의 체계이다.

  • R\RR 로 표기한다.

  • 실수체로 엄밀히 정의된다.

허수(imaginary number) : 제곱하여 −1-1−1 이 되는 수 −1\sqrt{-1}−1​ 이고 iii 라고 정의한다.

  • 실수는 제곱하였을 때 반드시 000 또는 양수가 되기 때문에 x2=−1x^2 = -1x2=−1 등의 방정식의 해를 표현할 수 없다. 그러므로 이와 같은 방정식의 해를 표현하기 위하여 제곱해서 음수가 되는 수를 고안하여 허수라고 칭한다.

  • 실제로 존재하는 수라고 여겨지지 않지만 특정 조건의 방정식의 해를 표현할 수 있게 해주어 수학적 가치가 있다고 여겨진다.

  • 허수가 알려지기 전까지 수는 111 차원적 개념에 머물렀지만 실수를 xxx 축으로 허수를 yyy 축으로 생각하여 복소평면이 만들어진 후 수의 개념이 222 차원으로 확장되었다. 또한 허수의 존재로 수가 nnn 차원으로 확장될 수도 있다는 가능성이 열렸다.

복소수(complex number) : 실수와 허수를 포괄하는 수이며 a+bi(a,b는실수)a + bi(a, b 는 실수)a+bi(a,b는실수) 의 꼴로 나타낸다.

  • aaa 를 실수부, bbb 를 허수부라고 한다.

  • 컬레복소수 : 복소수 a+bia+bia+bi 에서 허수부의 부호를 바꾼 복소수이다.

    • a+bi‾=a−bi\overline{a+bi} = a-bia+bi​=a−bi 로 표현한다.

사원수(quaternion) : 복소수를 확장한 체계이다.
