매개변수 방정식

곡선을 두 독립변수 x,yx, yx,y 로 나타내는 것이 아니라 매개변수를 통한 방정식으로 나타내면 훨씬 더 일반적인 곡선을 표현할 수 있다.

  • 매개변수 방정식(parametric equations) : x,yx, yx,y 가 각각 구간 III 의 ttt 에 대한 함수

    x=f(t),y=g(t) x = f(t), y = g(t) x=f(t),y=g(t)

    일 때, 위 방정식을 곡선의 매개변수 방정식이라 한다.

  • 매개변수 곡선(parametric curve) : 매개변수 방정식의 정의에서 매개변수 방정식으로 정의된 점들의 집합 (x,y)=(f(t),g(t))(x, y) = (f(t), g(t))(x,y)=(f(t),g(t)) 을 매개변수 곡선이라 한다.

  • 매개변수 구간(parametric interval) : 매개변수 방정식의 정의에서 정의역 III 를 매개변수 구간이라 한다.

  • 시점(initial point) : 매개변수 방정식의 정의에서 III 가 폐구간이고 ttt 가 a≤t≤ba \leq t \leq ba≤t≤b 로 정의되었을 때 점 (f(a),g(a))(f(a), g(a))(f(a),g(a)) 를 곡선의 시점이라 한다.

  • 종점(terminal point) : 매개변수 방정식의 정의에서 III 가 폐구간이고 ttt 가 a≤t≤ba \leq t \leq ba≤t≤b 로 정의되었을 때 점 (f(b),g(b))(f(b), g(b))(f(b),g(b)) 를 곡선의 종점이라 한다.

  • 매개변수 방정식 예시

    x=t2,y=t+1 x = t ^{2}, y = t+1 x=t2,y=t+1

    로 정의된 매개변수 방정식의 곡선을 그려보자.

    ttt 값에 따라 결정되는 x,yx, yx,y 의 값을 살펴보면 다음과 같다.

    이 그래프는 매개변수 방정식을 ttt 에 대하여 풀어서

    x=y2−2y+1 x = y ^{2} - 2y +1 x=y2−2y+1

    를 얻음으로써 그릴 수도 있다.

벡터

  • 양 : 어떤 물체의 크기이다.

    • 측정 가능한 양(물리량)과 측정 불가능한 양(비물리량)으로 나눌 수 있다.

  • 물리량 : 측정 가능한 양이다.

    • 벡터와 스칼라로 나눌 수 있다.

  • 비물리량 : 측정 불가능한 양이다.

    • 예시

      "AAA 의 키는 180180180 cm 이다." 에서 180180180 cm 의 키는 측정 가능한 물리량이다.

      반면 "AAA 가 BBB 보다 더 리더십이 있다." 에서 리더십의 비교는 측정이 불가능한 비물리량이다.

      그러나 리더십이라는 대상에 대한 단순화 및 집합으로의 환원을 거쳐서 측정하능하게 만들 수 있다는 가능성을 볼 수 있다. 여기에서 더 나아가 철학적 관념 또한 단순화와 수학적 모델링을 거쳐서 측정가능한 수로 격하시킬 수 있는 가능성도 엿볼 수 있다.

  • 스칼라(scala) : 크기만 있고 방향성이 없는 물리량이다.

    • 길이, 넓이, 부피 등이 스칼라량에 속한다.

    • 예시

      "AAA 는 202020 분을 뛰었다" 에서 202020 은 스칼라량이다.

  • 벡터(vector) : 크기와 방향성이 동시에 있는 물리량이다.

    • 스칼라량으로는 방향에 대한 정보가 부족하기에 방향에 대한 정보를 덧붙힌 것이 벡터이다.

    • 예시

      "AAA 는 202020 분을 뛰었다" 를 "AAA 는 동쪽으로 202020 분을 뛰었다" 로 바꾸면 방향성이 추가되었기 때문에 벡터량이다.

    • 역사

      물체에 두 힘을 동시에 가했을 때 물체의 힘의 크기를 나타내는 222 개의 선분이 이루는 평행사변형의 대각선을 그릴 수 있다. 이 대각선의 방향으로 그 물체가 움직이게 되는데 이러한 벡터의 합성을 스테빈(1548), 갈릴레오(1546) 가 발견했다.

      뉴턴(1642)은 속도, 힘 등을 방향이 있는 선분으로 나타내어 벡터의 기초를 만들었다.

      가우스(1777)는 복소수평면을 생각하여 복소평면에서 벡터를 나타낼 수 있게 하였다.

      해밀턴(1805)은 벡터에 좌표를 결합하여 벡터공간의 기초를 만들었다.

      그리스만(1809)은 벡터의 내적과 외적을 정의하였다.

      라플라스(1749)는 벡터해석학에 기여했다.

      기브스(1849), 헤비사이드(1850) 는 벡터해석학을 완성했다.

      이 벡터해석학은 19세기 물리학을 대표하는 전자기학의 필수적인 수학적 도구가 되었다. 또 현대에 와서는 의학과 공학에도 널리 쓰이는 중요한 개념이 되었다.

    • 다음과 같이 점 AAA 에서 점 BBB 로 방향이 주어진 선분 ABABAB 를 벡터 ABABAB 라 한다.

      • 이것을 형식언어로 AB→\overrightarrow{AB}AB 로 표현한다.

    • 다음과 같이 벡터를 시점과 종점이 아닌 단일 문자로 나타낼 수도 있다.

      a→,b→,c→,… \overrightarrow{a}, \overrightarrow{b}, \overrightarrow{c}, \dots a,b,c,…

    • 벡터는 행렬과 같이 수와 다른 개념이므로 행렬에서처럼 연산법칙, 항등원, 역원, 단위(수에서 111) 등을 새로 정의해야 한다.

  • 벡터의 시점(initial point) : 다음과 같이 점 AAA 에서 점 BBB 로 방향이 주어진 선분 ABABAB 를 벡터 ABABAB 에서

    점 AAA 를 벡터의 시점이라 한다.

  • 벡터의 종점(terminal point) : 다음과 같이 점 AAA 에서 점 BBB 로 방향이 주어진 선분 ABABAB 를 벡터 ABABAB 에서

    점 BBB 를 벡터의 종점이라 한다.

  • 스칼라 함수(scalar function) : 실숫값 함수와 같이 어떤 점을 하나의 스칼라에 대응시키는 함수이다.

    • 즉 여러 독립변수에 대응하는 하나의 함수 값이 스칼라가 된다.

    • 지금까지 살펴본 함수들이 스칼라 함수에 해당한다.

    • 실숫값 함수의 정의처럼 공역이 실수 집합 ℜ\Reℜ 인 함수

      f:X→ℜ f: X \to \Re f:X→ℜ

      이다.

    • 예시

      y=x2,y=sin⁡x,y=ln⁡x… y = x ^{2}, y = \sin x, y = \ln x \dots y=x2,y=sinx,y=lnx…

  • 벡터 함수(vector function) : 어떤 점을 하나의 벡터에 대응시키는 함수이다.

    • 여러 독립변수를 하나의 함수값, 즉 벡터에 대응시키는 함수이다.

    • 예시

      r(t)=<f(t),g(t),h(t)>=f(t)i+g(t)j+h(t)k r(t) = \bigg < f(t), g(t), h(t) \bigg > = f(t)i + g(t)j+h(t)k r(t)=⟨f(t),g(t),h(t)⟩=f(t)i+g(t)j+h(t)k

  • 장(field) : 공간에서 위치, 시간 등에 따라 그 성질을 다르게 나타내는 물리량이다.

    • 공간 내 점의 위치와 시간에 따라 다른 값을 가진다.

    • 공간의 모든 점에서 정의되는 물리량이다.

      • 물리량이므로 스칼라량, 벡터량을 지닌다.

  • 스칼라 장(scalar field) : 공간 내 각 점의 위치, 시간 등에 따라 결정되는 크기를 갖는 스칼라를 나타내는 분포이다.

    • 즉 공간 내 각 점에 물리적으로 스칼라 값을 대응시키는 스칼라 함수이다.

    • 예시

      대기의 각 점의 온도, 밀도, 압력

    • 예시

      전하가 있는 공간에서의 전위의 분포

  • 벡터 장(vector field) : 공간 내 각 점의 위치, 시간 등에 따라 결정되는 크기와 방향을 갖는 벡터를 나타내는 분포이다.

    • 예시

      속도장, 역장(力場)은 크기 뿐만 아니라 방향을 가지므로 벡터장에 해당한다.

      • 속도장은 각 점에서 속도 벡터를 대응시킨 것으로 해류 속도장, 풍속 속도장 등이 있다.

      • 역장은 각 점에서 힘 벡터를 대응시킨 것으로써 중력장, 전기장, 자기장 등이 있다.

        • 일례로 중력장은 중력이 미치는 공간 내 각 점에 중력 벡터를 대응시킨 벡터 함수이다.

여러가지 벡터

  • 영벡터(zero vector) : 임의의 점 AAA 에 대하여 AA→\overrightarrow{AA}AA 와 같이 시점과 종점이 일치하는 벡터를 영벡터라 한다.

    • 기호로 0→\overrightarrow{0}0 으로 나타낸다.

    • 영벡터는 크기가 000 고 방향은 없다고 정의한다.

    • 실수연산에서 항등원 역할을 한다.

  • 역벡터(inverse vector) : 벡터 AB→\overrightarrow{AB}AB 에 대하여 크기는 같고 방향이 반대인 벡터 BA→\overrightarrow{BA}BA 를 벡터 AB→\overrightarrow{AB}AB 의 역벡터라 한다.

    • AB→\overrightarrow{AB}AB 역벡터를 −-− 를 붙혀서 −AB→-\overrightarrow{AB}−AB 로 나타낸다.

      BA→=−AB→ \overrightarrow{BA} = -\overrightarrow{AB} BA=−AB

    • 실수연산에서 역원 역할을 한다.

  • 단위벡터(unit vector) : 크기가 111 인 벡터이다.

    • 실수연산에서 111 의 역할을 한다.

  • 평면벡터(plane vector) : 평면에서 정의된 벡터이다.

  • 공간벡터(space vector) : 공간에서 정의된 벡터이다.

실수에서 했던 연산들을 벡터에 대하여 재정의하고 있는 것이다.

  • 벡터의 크기 : 벡터 AB→\overrightarrow{AB}AB 에서 선분 ABABAB 의 길이를 벡터의 크기라 한다.

    • 형식언어로 ∣AB→∣|\overrightarrow{AB}|∣AB∣ 로 나타낸다.

    • 다음과 같이 벡터의 크기를 시점과 종점이 아닌 단일 문자로 나타낼 수도 있다.

      ∣a→∣,∣b→∣,∣c→∣,… |\overrightarrow{a}|, |\overrightarrow{b}|, |\overrightarrow{c}|, \dots ∣a∣,∣b∣,∣c∣,…

  • 두 벡터가 같다 : 시점과 종점의 위치에 관계 없이 두 벡터 a→,b→\overrightarrow{a}, \overrightarrow{b}a,b 의 크기와 방향이 같으면 두 벡터 a→,b→\overrightarrow{a}, \overrightarrow{b}a,b 를 같다고 한다.

    • 두 벡터 a→,b→\overrightarrow{a}, \overrightarrow{b}a,b 가 각각 선분 AB‾,CD↔\overline{AB}, \overleftrightarrow{CD}AB,CD 로 구성되었을 때 이 둘이 같다는 것을

      a→=b→⇔AB→=CD→ \overrightarrow{a} = \overrightarrow{b} \Leftrightarrow \overrightarrow{AB} = \overrightarrow{CD} a=b⇔AB=CD

      로 표현한다.

    • 예시

      위 세 벡터는 같은 방향과 같은 길이를 지니므로 모두 같다.

      즉, 왼쪽부터 차례로 AB→,CD→,OP→\overrightarrow{AB}, \overrightarrow{CD}, \overrightarrow{OP}AB,CD,OP 라고 한다면

      AB→=CD→=OP→ \overrightarrow{AB}= \overrightarrow{CD}= \overrightarrow{OP} AB=CD=OP

      이다.
